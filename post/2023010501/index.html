搭建算法刷题环境(C++版)

2023/01/05

文章目录

    • 环境前置要求
    • 安装并配置Docker Container
    • 配置VSCode
    • Show Me Code

环境前置要求

  1. 需要先安装好 Docker，Docker 的好处不言而喻，方便跨平台，也避免污染宿主机环境，起到隔离的作用。
  2. 用什么操作系统都无所谓，毕竟 Docker 跨平台。本人用的 mac 。
  3. 代码编辑器的话选择 VSCode 就好，轻量级、插件多，好用不必多说。

安装并配置Docker Container

docker pull ubuntu

docker run -itd --name ubt-cpp -v ~/code:/data ubuntu

启动 Container 后，进入容器:

docker exec -it ubt-cpp bash

安装C++必要环境：

apt update
apt install gcc g++ gdb make

clang-format 也可以装下，可以按照自己的风格格式化代码：

apt install clang-format

我喜欢 Google 的C++ 代码风格，所以我的 .clang-format 文件配置如下：

---
Language:        Cpp
BasedOnStyle:  Google
AccessModifierOffset: -4
ConstructorInitializerIndentWidth: 4
ContinuationIndentWidth: 4
IndentWidth:     4
TabWidth:        4
UseTab:          Never
...

配置VSCode

  1. 首先配置的是C/C++插件，在插件管理器搜索 C++ 即可。
  2. 为了使 VSCode 连接 Container，需要安装Dev Containers插件，在插件管理器搜索 Container 即可。

Show Me Code

进入 Container 内，执行如下命令：

mkdir -p /data/ltcode
cd /data/ltcode

创建几个必要文件，目录结构如下：

.
|-- .clang-format
|-- Makefile
|-- common.h
|-- input.data
|-- main
|-- main.cpp
`-- output.data

Makefile 是 make 的配置文件，common.h 是常用的头文件和一些数据结构和工具函数的定义，input.data 是数据输入文件，main 是编译出的二进制可执行文件，main.cpp 是算法实现的主要地方，output.data 是数据输出文件。

这里有一个小的Demo可以参考下：

Makefile 内容如下：

.PHONY: build
build:
	g++ -std=c++11 -o main main.cpp

.PHONY: run
run:
	./main

.PHONY: run_with_data
run_with_data:
	./main < input.data > output.data

common.h 内容如下：

#ifndef _COMMON_H_
#define _COMMON_H_

// common include
#include <algorithm>
#include <iostream>
#include <queue>
#include <stack>
#include <string>
#include <unordered_map>
#include <vector>

// common class and struct

// common function

#endif

main.cpp 的代码示例如下：

#include "common.h"

using namespace std;

int main() {
    int a = 0, b = 0;
    while (cin >> a >> b) {
        cout << a + b << endl;
    }
    return 0;
}

代码比较简单，逻辑是输入两个整数 a 和 b，求它们的和并输出。

测试 input.data 数据如下：

11 13
1233 4432
23 999

首先编译代码，命令行执行make build。

然后执行代码，命令行执行make run_with_data。

执行完后，可以看到输出文件 output.data 有如下内容：

24
5665
1022

这样，使用 C++ 刷算法题的环境就搭建好了，后面只要修改 common.h 和 main.cpp 文件内容即可。
